# 资源浏览器 - 使用指南

## ✅ 新设计 - 文件树浏览方式

已将"JSX 预览器"改造为**资源浏览器**,采用更直观的文件树浏览方式。

## 🎯 核心功能

### 1. **文件树导航**
左侧显示 `assets/` 目录的完整文件结构:
- 📁 avatars/ - 头像资源
- 📁 icons/ - 图标资源  
- 📁 backgrounds/ - 背景资源
- 📁 decorations/ - 装饰元素

### 2. **一键查看**
点击任意文件即可查看:
- SVG 文件
- JSX 组件文件
- 代码高亮显示

### 3. **切换视图**
- **代码视图** - 查看源代码(黑色主题,代码高亮)
- **预览视图** - 直接渲染 SVG(仅 SVG 文件支持)

### 4. **快速复制**
点击"复制"按钮,一键复制文件内容到剪贴板

## 📂 当前可浏览的文件

## 🎨 使用方法

### 基本操作

1. **打开资源浏览器**
   - 点击顶部导航的"资源浏览器"

2. **浏览文件树**
   - 左侧显示所有文件夹和文件
   - 点击文件名加载内容

3. **切换视图**
   - 点击"代码"按钮查看源代码
   - 点击"预览"按钮(SVG 文件)查看渲染效果

4. **复制代码**
   - 点击"复制"按钮
   - 代码已复制到剪贴板

### 示例工作流

```
1. 打开资源浏览器
2. 点击 decorations/rainbow-cloud.svg
3. 查看代码结构
4. 点击"预览"查看效果
5. 点击"复制"获取代码
6. 在你的组件中使用
```

## 💡 优势

相比之前的 JSX 预览器:

| 旧方案 | 新方案 |
|-------|-------|
| 需要手动输入代码 | 直接点击文件加载 |
| 容易出现语法错误 | 显示真实文件内容 |
| 只能预览模板 | 可以浏览所有资源 |
| 需要记住文件路径 | 可视化文件树 |

## 🎯 典型使用场景

1. 打开资源浏览器

1. 打开资源浏览器
2. 浏览 `decorations/` 目录
3. 切换到预览视图
4. 快速查看所有装饰元素效果

### 场景 4: 复制组件代码
1. 打开资源浏览器
2. 选择 `backgrounds/scenes.jsx`

4. 复制到你的项目中使用

## 🔧 技术实现

### 文件树结构
```javascript
const fileTree = {
  avatars: [
    { name: 'doctor-avatar.svg', type: 'svg', path: '/src/assets/avatars/...' }
  ],
  icons: [...],
  backgrounds: [...],
  decorations: [...]
}
```

### 异步加载
```javascript
const loadFile = async (file) => {
  const response = await fetch(file.path)
  const text = await response.text()
  setFileContent(text)
}
```

### 视图切换
- **代码视图**: `<pre>` 标签 + 语法高亮
- **预览视图**: `dangerouslySetInnerHTML` 渲染 SVG

## 📝 界面说明

### 左侧面板 (文件树)
- 📁 文件夹图标
- 📄 文件图标 (SVG/JSX)
- 高亮显示当前选中文件
- 悬停效果

### 右侧面板 (内容显示)
- 顶部工具栏:
  - 文件名和路径
  - 视图切换按钮
  - 复制按钮
- 内容区:
  - 代码显示 (黑色主题)
  - 或 SVG 预览

### 空状态
未选择文件时显示:
- 大文件夹图标
- 提示文字:"选择一个文件"
